### Requirements

- Python 3.8 or newer  
- OpenCV
- NumPy  
- scikit-fuzzy  

### Build & Run

1. **Clone Repository**

       git clone https://github.com/ichsanyudika12/re305-webots-fuzzy-tracker.git
       
       cd re305-webots-fuzzy-tracker

2. **Open the world file**

       worlds/world.wbt

3. **Set the e-puck robot controller**

       tracking.py
